# 中文通話紀錄斷詞/斷句工具測試

## 專案目的
本專案用於測試兩種中文 NLP 工具在通話紀錄資料上的處理效果，並比較其輸出結果：

- spaCy
- ckip-transformers

測試資料為通話內容紀錄。為避免真實個資外洩，原始通話紀錄中的個人資訊已使用 `假個資/` 資料夾中的內容進行替換後，再進行後續測試。

---

## 測試內容
本次測試主要觀察以下項目：

- 中文文本的斷詞/斷句表現
- 命名實體辨識（如姓名、地點、組織等）結果
- 在通話紀錄場景下的可讀性與實用性
- 視覺化輸出結果的差異

---

## 資料處理流程
1. 準備原始通話紀錄資料
2. 使用 `假個資/` 中的假資料替換原始內容中的敏感資訊
3. 分別將處理後文本輸入：
   - `spaCy`
   - `ckip-transformers`
4. 將各自的分析/可視化結果輸出為圖檔
5. 將輸出結果依工具分類儲存
